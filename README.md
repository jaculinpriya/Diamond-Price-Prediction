# Diamond-Price-Prediction
To predict the diamond price in INR and USD, sample attributes used like carat, cut, clarity, and color to decide pricing. Did EDA &amp; Data Visualization, Feature Engineering, Outlier &amp; Skewness Handling, Regression using ML algorithms, including ANN, K-Means Clustering &amp; Cluster Labeling, Dimensionality Reduction with PCA and output in Streamlit UI.
